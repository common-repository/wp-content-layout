== Description ==

Simple WordPress layout plugin, with the help of this plugin you can create a layout for your content and easily can display on your page or post.You can choose Grids in 1,2, 3, 4 column. Select your desired layout and use shortcode to display the grid content on your page post or any where. 
You can create any number of the rows or layout and you find short code for the every layout .WP content layout is also responsive so your content will be any device friendly.You can use this plugin for google adsense, adsense or display also the any adds or any thing on your page and display any where using the short code.
